      一些简单的说明 #

      每家厂商提供的vps产品说明不太一致，比如有些产品会列出cpu型号，有些则不会；有些vps只提供一个ipv4， 而有些则会提供多个， 还有的只提供ipv6，因此购买时，需要注意一下。这里简单的做下说明： #

      • cpu 如果没有带型号，一般是比较低配的Intel型号，例如1x、1 这种， 默认是低配的cpu，如果给出了型号，例如：1x AMD Ryzen 9 3900x，则一般可以认为这是相对较好的cpu。

      • 硬盘 默认使用普通SSD，普通SSD 优于 HDD；NVMe SSD 优于SSD。（NVMe SSD > SSD > HDD）

      • 内存 DDRx 内存优于普通内存（DDRx > 普通）

      • ipv4 如果大于等于1个，说明提供了ipv4, 如果是0个，那表示只提供了ipv6, 需要特别注意

      以上这些信息，从价格上也能反映出来，比如cpu核数、内存等都基本一致时，cpu好的价格明显好于cpu差的。
